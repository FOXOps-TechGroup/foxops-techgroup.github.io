竞赛环境 Environment

选手机环境参考。 Refer to the mobile environment.

系统

  • Ubuntu 24.04.3 LTS with Linux Kernel 6.14.0.29-generic；

桌面环境

  • Xfce 4.18;

编辑器

  • vi/vim
  • gvim
  • emacs
  • gedit
  • geany
  • kate

支持的语言环境

  • Java
    • Openjdk version 21.0.8
    • 编译参数: -encoding UTF-8 -sourcepath . -d . {files}
    • 运行参数: -Dfile.encoding=UTF-8 -XX:+UseSerialGC -Xss64m -Xms1920m -Xmx1920m
  • C
    • gcc 版本 14.2.0
    • 编译参数: -x c -g -O2 -std=gnu23 -static {files} -lm
  • C++
    • g++ 版本 14.2.0
    • 编译参数: -x c++ -g -O2 -std=gnu++23 -static {files}
  • Python 3
    • PyPy3 version 7.3.18 with GCC 10.2.1 providing Python 3.11.11
    • 关于Python 3支持的模块与库，详见附录。
  • Kotlin
    • Version 1.9.24
    • compiler flags: -d . {files}
    • runtime flags: -Dfile.encoding=UTF-8 -J-XX:+UseSerialGC -J-Xss64m -J-Xms1920m -J-Xmx1920m
